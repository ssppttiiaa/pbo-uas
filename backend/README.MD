# SmartBudget Rumah Tangga - Backend

Backend aplikasi **SmartBudget Rumah Tangga** menggunakan **Python Flask** dan **Supabase** sebagai database.

---

## Teknologi

- Python
- Flask
- Supabase
- PostgreSQL
- JWT
- bcrypt
- python-dotenv

---

## Instalasi

```bash
pip install -r requirements.txt
```

---

## Menjalankan Project

```bash
python app.py
```

---

## Endpoint

### Authentication

- POST /register
- POST /login

### User

- GET /profile
- PUT /profile
- DELETE /profile
